# Reusable Marketing Image Prompt Pack

## Purpose And Honesty Guardrails

This is a reusable marketing prompt/reference pack for Ember seek-and-find books. It is built from book prompts, source plans, character references, and cover/listing notes, not from finished interior pages.

Fresh AI marketing art is appropriate for:

- Social posts
- Teasers
- Coming-soon graphics
- Character promo
- Launch graphics
- Website or social banners

Use real book assets for:

- Amazon listing preview images
- "Inside the book" previews
- Anything claiming to show a sample page
- Paid ads after launch
- Any image that could be mistaken for a real cover, page, spread, or mockup

Fresh art must not misrepresent the actual book. Do not advertise hidden objects, pages, features, difficulty, sample pages, side characters, or book worlds that are not actually supported by the finished book.

Sparkleflame Festival language is Book 1-specific. Do not treat Sparkleflame as the series brand. The broader series naming should stay flexible, using a working pattern like:

```text
Ember the Dragon and [Book Title]
```

## Marketing Text Policy

Do not use vague instructions like "leave space for text" or "text added later" when a marketing image is supposed to include copy.

For every marketing prompt:

- If text should appear, the prompt must list the exact words to render.
- If text should not appear, the prompt must say `No text.` clearly.
- Do not allow extra words, fake text, gibberish text, misspellings, labels, watermarks, page numbers, or decorative unreadable writing.
- If the image generator cannot render the exact approved words cleanly, regenerate from the same exact-text prompt or rebuild the image using real editable typography outside the generator.

## Book 1 Visual Clarity And Placement Rules

Use these rules for Book 1 seek-style marketing images and any Book 1 prompt adapted from the source plan:

- Make the image rich but not overstuffed. Use fewer total objects than a finished interior seek page.
- Build clear clusters: place named mission items, main find/category items, and decorative filler in separate intentional zones.
- Put the named mission item in a fair hiding spot on a believable surface, tucked edge, shelf, stall, garden bed, stone cluster, or lantern/string area. Do not float it randomly.
- Place broad category items where they naturally belong: bakery items on counters/trays, market items in stalls/baskets, trail items along edges/rocks, garden items in beds/leaves, cottage items on shelves/tables.
- Keep walkways, paths, bridges, trails, door thresholds, and the viewer's eye path mostly open and readable. Add only a few small edge details there.
- Avoid random scatter, floor litter, muddy clutter, and dense repeated tiny props that make the search feel accidental instead of designed.
