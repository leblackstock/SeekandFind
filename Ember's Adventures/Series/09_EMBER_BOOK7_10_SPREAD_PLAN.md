# Ember Book 7 10-Spread Plan

Title: **Ember and the Cozy Snow Village Search**. Use with `02_EMBER_ACTIVE_RULESET.md`.

## Production Order

Printed order is story/list left page then seek image right page. Production order is seek image first, then story/list page after the find list is approved:

```text
1, 3, 2, 5, 4, 7, 6, 9, 8, 11, 10, 13, 12, 15, 14, 17, 16, 19, 18, 21, 20, 22-23, 24
```

## Character Use

Ember is the recurring guide on every seek page. Use Ember reference images `Ember-001`, `Ember-002`, and `Ember-003`.

Book 7 named supporting characters must come from canon only: Pebblekins for snowy stone/cottage pages, Hootiepuff for night clue pages, Luma Leafwhisk for evergreen pages, Gemma Glint for the ice-gem workshop, and Elder Glowkeeper for the village clock/finale. Do not introduce new named side characters.
