# Visual Style Guide

Use polished soft 2.25D children's storybook illustration style with lightly illustrated edges, rounded plush-like forms, gentle sculpted volume, painterly shading, warm cheerful lighting, smooth clean surfaces, and magical but readable staging.

Do not use flat cartoon styling, photorealism, overly 3D rendering, scary lighting, muddy clutter, or protected brand imitation.

For seek pages, build clear foreground, midground, and background zones. Place objects where they naturally belong. Keep paths, bridges, door thresholds, and the viewer's eye path mostly open and readable.

For marketing art, keep the image clear at phone size and make sure it does not pretend to be a real interior page, real cover, listing preview, or sample-page asset unless real book assets are used.

# No Readable Text Policy

## Interior Image Prompts

Do not request readable generated text inside AI-generated seek-page images. No labels, signs, page numbers, watermarks, fake logos, gibberish text, decorative unreadable writing, or accidental lettering.

Story/list pages, cover typography, listing graphics, and marketing posts should use real editable layout text unless the user explicitly approves exact text generation for a marketing image.

For every marketing prompt:
