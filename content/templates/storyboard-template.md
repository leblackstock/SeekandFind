# Storyboard Template

Use 3-5 timed beats for a default 15-second Seedance storyboard. Include reference guidance, camera movement, character motion, scene continuity, CapCut edit notes, music/sound suggestion, and manual upload checklist.

Important workflow split:

- Motion-only image-to-video storyboards should use the image-to-video prompt and must avoid generated captions, labels, or readable text.
- CapCut storyboard/script blocks create visible bottom text on the video. Use that path only for captioned narration shorts or description-led Shorts/Reels/TikToks.
- Captioned narration shorts may also become an AI-animation workflow later, but that path has not been worked through yet.
- Do not paste timed beat descriptions into CapCut storyboard/script blocks when the goal is a clean no-text character animation.
